# vitæ

My curriculum vitæ.  To generate a PDF version, run `make` or compile
using latexmk:

    latexmk -norc -pdf -ps- -dvi- vitae.tex

Using make would automatically generate the `gitHeadInfo.gin` file
required by the [gitinfo2][1] package.
